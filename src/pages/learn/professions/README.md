# Learn Professions

## Purpose

This section groups learning content by role so people can see why a concept matters to their work, not just what the term means in isolation.

## Route ownership

- `/learn/professions`

## What belongs here

- Role-oriented learning entry points for machinists, quality teams, supervisors, manufacturing engineers, robotics/automation technicians, and other shop roles

## Where to start

- `LearnProfessions.tsx` for the section landing page

## Content rules

- Explain job relevance before product relevance
- Reuse existing profession taxonomy where possible instead of inventing parallel labels
- Cross-link into fundamentals, glossary, and tutorials when a role needs more depth
