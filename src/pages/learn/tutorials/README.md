# Learn Tutorials

## Purpose

This section owns named tutorial ecosystems and high-intent how-to clusters that can attract search traffic on their own.

## Route ownership

- `/learn/tutorials`

## What belongs here

- Tutorial hubs for OpenClaw, Hermes, NemoClaw, and related installation/how-to/troubleshooting content
- Search-intent landing pages that need deeper walkthrough structure than glossary or fundamentals pages

## Where to start

- `LearnTutorials.tsx` for the tutorial landing page and cluster overview
- Child directories for OpenClaw, Hermes, and NemoClaw install/search-intent pages

## Content rules

- Prefer original tutorial structure and screenshots over copied documentation
- Organize by real search intent such as install, setup, troubleshooting, and workflow walkthroughs
- Link back to fundamentals and role-based pages when a tutorial assumes background knowledge
- Include a clear JobLine.ai bridge area so readers can move from public learning into platform evaluation or workflow use without turning the page into a hard sell
