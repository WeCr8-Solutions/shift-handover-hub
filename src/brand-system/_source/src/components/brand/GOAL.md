# GOAL — Brand Components

**Objective:** Provide a single, canonical implementation of the JobLine.ai visual identity marks that can be used across the app, flyer previews, print exports, and any connected system.

**Success criteria:**
- [ ] Logo renders identically to print collateral at all sizes (scale 0.5–3x)
- [ ] Mark and logo can be used independently
- [ ] No hardcoded colors — all values from `tokens/colors.ts`
- [ ] Works in React Native and React DOM (no native-only APIs)

**Owner:** WeCr8 Solutions design system
**Priority:** P0 — dependency of all other components
