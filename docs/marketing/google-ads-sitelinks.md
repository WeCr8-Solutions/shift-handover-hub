# Google Ads — Account-Level Sitelinks

> **Use:** Paste each block into Google Ads → Assets → Sitelinks → **New sitelink**.
> All character counts are pre-verified against Google's limits
> (Text ≤25, Description 1 ≤35, Description 2 ≤35).
> Final URLs use `https://jobline.ai` so they match the verified domain.
>
> **Pick at least 6** to satisfy the "Add at least 6 sitelinks" recommendation;
> loading all 10 lets Google rotate the best performers.
