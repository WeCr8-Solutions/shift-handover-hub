# JobLine.ai — Growth Strategy Playbook

Internal, high-level strategy library for getting the brand and product in front of the right people on both sides of the marketplace: **manufacturing shops** (demand) and **operators/talent** (supply).

This is a **strategy** library, not a tactical SOP collection. Each doc explains *why* and *what posture to take* — execution detail (templates, ad copy, posting schedules) lives elsewhere or with the teammate running the channel.

Pairs with `docs/investors/` so fundraising narrative and GTM narrative stay in lockstep.

## How to use this library

- **Quarterly review**: re-read 00, 08, 09 every quarter. Update if positioning, KPIs, or priorities shifted.
- **Before launching a channel**: read the relevant doc (02/03/04/05/06/07) and confirm the play fits the stated posture. If it doesn't, either change the play or update the strategy — don't run both in contradiction.
- **Before a fundraise update**: reconcile positioning language with `docs/investors/one-pager.md`. The two should never disagree.
- **When in doubt**: ship work that compounds (content, credentials, partnerships) over work that doesn't (paid ads at this stage).

## What this library is NOT

- Not a campaign calendar.
- Not an ad-creative or email-template repo.
- Not a substitute for talking to customers.
- Not a list of features — features serve strategy, not the other way around.
