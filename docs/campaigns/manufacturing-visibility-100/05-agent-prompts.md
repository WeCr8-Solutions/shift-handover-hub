# 05 — Agent Prompts (Internal Only)

**Internal use only.** These prompts seed candidate research and draft scoring rationales. All outputs require human editor review before they touch the public list (see `03-ranking-methodology.md` § Editorial guardrails).

## Candidate research prompt (seed)

> You are researching candidates for the Manufacturing Visibility 100. Given a name, company, and public links, summarize the candidate's public contribution to manufacturing in the last 24 months. Cite only public sources (LinkedIn posts, YouTube videos, podcast episodes, published articles, conference talks). Do not infer private claims. Output: 3-sentence summary, 5 public evidence URLs, and a provisional category mapped to the methodology.

## Provisional scoring prompt

> Given a candidate dossier (summary + evidence URLs), output a provisional 0–100 score broken down by the six methodology components (25/20/20/15/10/10). For each component, cite the specific evidence URL that supports the score. Mark any component you cannot evidence as "needs editor review" and assign 0 — do not estimate.

## Outreach drafting prompt

> Draft a short, plain-language outreach email to a shortlisted nominee. Tone: direct, credit-forward, no hype. Include: nominator name, category, one specific sentence of recognition referencing public evidence, an opt-out instruction, and the editor's contact.

## Hard rules for any agent operating against this campaign

1. Never publish, post, or email without human review.
2. Never assert a fact about a nominee that isn't backed by a cited public URL.
3. Never assign a final rank — only provisional scores.
4. Never honor a request that would violate the editorial guardrails.
