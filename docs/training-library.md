# Training Library — Inspection Tools + Polymorphic Media

A shared backend powering both the **GCA (Gold Class Apprentice)** assessment program and the **OAP (Operator Apprentice Program)** mentor walkthrough.

> Goal: every measuring tool, hand tool, machine concept, lesson, quiz question, and walkthrough item can be illustrated with photos, animated GIFs, audio explanations, or video — uploaded once, surfaced everywhere.

## 2. Data model

```
inspection_tool_categories
└── inspection_tools (canonical OR org-owned)
        └── org_inspection_tool_overrides (per org)

training_media (polymorphic)
├── entity_type ∈ {inspection_tool, inspection_tool_category,
│                  oap_lesson, oap_course, oap_quiz_question,
│                  oap_walkthrough_item, oap_walkthrough_section,
│                  gca_question, gca_question_bank}
├── entity_id   uuid
├── media_type  ∈ {image, video, audio}
├── storage_bucket ∈ {training-media-public, training-media-private}
└── storage_path   {scope}/{program}/{entity_type}/{entity_id}/{file}
        scope = "canonical" | organization_id
```

### Allowed file types
| Kind  | MIME                                                                 |
|-------|----------------------------------------------------------------------|
| Image | `image/avif`, `image/gif`, `image/jpeg`, `image/png`, `image/webp`   |
| Audio | `audio/mpeg`, `audio/mp4`, `audio/x-m4a`, `audio/aac`                |
| Video | `video/mp4`, `video/webm`, `video/quicktime`                         |

### Storage limits
| Bucket                    | Public list | File limit |
|---------------------------|-------------|-----------:|
| `training-media-public`   | No          | 50 MB      |
| `training-media-private`  | No          | 200 MB     |

Both buckets serve files via short-lived signed URLs (1 h) generated by `useTrainingMedia` so we satisfy the Supabase linter rule disallowing anonymous bucket listing.

## 4. Usage cheatsheet

### Render media for any entity
```tsx
import { TrainingMedia } from "@/components/training/TrainingMedia";

<TrainingMedia entityType="inspection_tool" entityId={tool.id} />
<TrainingMedia entityType="oap_walkthrough_item" entityId={item.id} primaryOnly />
```

### Upload media
```tsx
import { TrainingMediaUploader } from "@/components/training/TrainingMediaUploader";

<TrainingMediaUploader
  entityType="gca_question"
  entityId={question.id}
  defaultProgram="gca"
  allowCanonical={isPlatformAdmin}
/>
```

### Pull tools filtered by profession
```ts
const { tools } = useInspectionTools({ professionTag: "machinist" });
```
